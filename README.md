# KoinX – Tax Loss Harvesting

## Live Demo
> Add your Vercel link here after deploying

## Local Setup
```
npm install
npm start
```
Opens at http://localhost:3000

## Features
- Dollar ($) currency display
- How it works section with 3-step guide
- Pre & After Harvesting cards (live update)
- Holdings table with expandable rows (click ▼ arrow)
- Select/Deselect all holdings
- View All toggle (5 shown by default)
- Tax savings banner
- Mobile responsive
- Skeleton loaders & error states

## Deploy
Upload the contents of this folder to a GitHub repo, then connect to Vercel.
Vercel will auto-detect Create React App and deploy correctly.
